# ServiceMate – Local Service Booking Platform

ServiceMate is a local service booking platform that connects customers with nearby service providers such as electricians, plumbers, carpenters, and cleaners. The platform allows users to easily discover services, book appointments, and manage service requests in one place.

---

## 🚀 Project Objective

The goal of ServiceMate is to simplify the process of finding and booking trusted local services by providing a centralized digital platform for customers and service providers.

---

## 🛠️ Tech Stack

Frontend: React.js  
Backend: Spring Boot   
Database: MySQL 
API Testing: Postman  
Version Control: Git & GitHub  

---

## 📌 Key Features

- User Registration and Login
- Role-Based Access (Customer / Service Provider / Admin)
- Browse Available Services
- Book Local Service Providers
- Manage Service Requests
- Review and Rating System
- Secure Authentication

---

## 📂 Database Entities

The system database includes the following core tables:

- Users
- Service Providers
- Services
- Bookings
- Reviews
- Payments

These tables are connected using primary and foreign key relationships to maintain data integrity.

---

## 🔐 Authentication Module

The authentication system provides:

- User Signup API
- User Login API
- Input validation
- Secure authentication handling
- Database connectivity verification

All APIs are tested using Postman.
